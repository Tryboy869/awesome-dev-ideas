# Code de Conduite

## Notre engagement

Nous nous engageons à fournir un environnement accueillant, inclusif et exempt de harcèlement pour tous, indépendamment de :

- L'âge, la taille corporelle, le handicap, l'ethnicité
- L'identité et l'expression de genre
- Le niveau d'expérience ou d'éducation
- La nationalité, l'apparence personnelle, la race, la religion
- L'identité et l'orientation sexuelle

**Ceci est un espace pour la créativité technique et le discours intellectuel honnête.** Nous valorisons les idées audacieuses, la critique constructive et le désaccord respectueux.

---

## Nos standards

### ✅ Comportements attendus

**Critiquer les idées, pas les personnes :**
- "Cette approche a un défaut dans le mécanisme de consensus" ✅
- "Tu ne comprends clairement rien aux systèmes distribués" ❌

**Présumer la bonne foi :**
- Tout le monde est ici pour apprendre et partager
- Les malentendus arrivent — clarifiez avant d'escalader
- Tous les désaccords ne sont pas des attaques

**Être constructif :**
- Si vous voyez un problème, suggérez une solution
- Partagez des recherches ou des références pertinentes
- Reconnaissez quand vous n'êtes pas certain

**Célébrer la créativité :**
- Les idées "folles mais cohérentes" sont les bienvenues
- Les expériences échouées nous apprennent quelque chose
- Encouragez l'expérimentation et l'itération

---

### ❌ Comportements inacceptables

Nous avons une **tolérance zéro** pour :

- **Attaques personnelles** — insulter, rabaisser ou se moquer des contributeurs
- **Harcèlement** — commentaires non sollicités sur les caractéristiques personnelles
- **Trolling** — faire délibérément dérailler les discussions ou poster de mauvaise foi
- **Doxxing** — partager des informations privées sans consentement
- **Spam** — contenu promotionnel, flooding hors-sujet, ou absurdités générées par IA

**Exemples de violations :**

- "Cette idée est stupide et toi aussi" ❌
- "Typique de la pensée [groupe]" ❌
- Rejeter systématiquement une critique technique valide ❌
- Poster des dizaines d'idées de faible qualité générées par IA ❌

## Application

### Système d'avertissement

1. **Première infraction :** Avertissement public dans le fil de discussion
2. **Deuxième infraction :** Avertissement formel par email/DM
3. **Troisième infraction :** Bannissement permanent du dépôt

**Violations graves (menaces, doxxing, discours haineux) = bannissement immédiat.**

---

### Signalement

Si vous êtes témoin ou victime d'une violation du Code de Conduite :

**Option 1 : Signalement public**
- Commentez dans le fil : `@anzize Ceci viole le Code de Conduite`
- Les mainteneurs enquêteront publiquement

**Option 2 : Signalement privé**
- Email : contact@proton.me
- DM GitHub : [@anzize](https://github.com/anzize)
- Les signalements sont confidentiels

**Que se passe-t-il ensuite :**
1. Les mainteneurs examinent le signalement sous 48 heures
2. Si confirmé, le contrevenant est averti ou banni
3. Le rapporteur est informé du résultat

---

## Périmètre

Ce Code de Conduite s'applique à :

- Toutes les interactions du dépôt (Issues, PRs, Discussions)
- Communication dans les canaux liés (Twitter, Discord si créé)
- Toute communication publique ou privée référençant /dev/ideas

Il ne s'applique **pas** à :
- Le contenu des idées techniques elles-mêmes (les idées peuvent être controversées)
- Les désaccords sur ce qui constitue du "bullshit" (c'est subjectif)

---

## Maintenir les standards

**Les mainteneurs sont censés :**

- Modéliser un comportement respectueux
- Appliquer ce Code de manière juste et cohérente
- Se récuser des cas impliquant des conflits personnels

**Les contributeurs peuvent aider en :**

- Signalant les violations
- Désamorçant les discussions houleuses
- Accueillant les nouveaux contributeurs

---

## Philosophie

> **C'est un espace où les gens peuvent partager des idées "absurdes mais cohérentes" sans craindre l'humiliation.**

Nous rejetons à la fois :
- Le gatekeeping toxique qui étouffe la créativité
- L'acceptation inconditionnelle d'idées techniquement défaillantes

**L'équilibre, c'est la rigueur intellectuelle + la bienveillance humaine.**
